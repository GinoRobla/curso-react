# Master en React: ReactJS, Hooks, MERN, NodeJS, JWT+

  🚀 Este repositorio contiene los proyectos desarrollados durante el curso "Master en React: ReactJS, Hooks,
  MERN, NodeJS, JWT+", impartido por Víctor Robles.

  En él se construyen 3 aplicaciones web completas utilizando el stack MERN, desarrollando tanto el front‑end       
  con React (usando Hooks, Context y React Router) como el back‑end con Node.js, Express y MongoDB, incluyendo      
  autenticación basada en JWT.

  ## 🌟 DEMO EN VIVO

  ### 🎬 Catálogo de Películas
  **Demo en vivo**: [https://curso-react-mauve-omega.vercel.app](https://curso-react-mauve-omega.vercel.app)        

  ## 📁 Proyectos Incluidos

  ### 1. Catálogo de Películas (React + Vite)
  **Descripción**: Aplicación para gestionar un catálogo personal de películas con funciones CRUD y
  persistencia local.
  **Tecnologías**:
  - React 19 + Hooks
  - Vite
  - localStorage
  **Características**:
  - Agregar, editar y eliminar películas
  - Búsqueda en tiempo real
  - Persistencia local en el navegador
  - Interfaz responsive

  ### 2. Portafolio Personal (React)
  **Descripción**: Aplicación SPA que muestra proyectos, habilidades y contacto. Permite navegación con React       
  Router y manejo de estado.
  **Tecnologías**:
  - React
  - Hooks (useState, useEffect, useContext, useReducer)
  - React Router
  **Características**:
  - Navegación SPA
  - Gestión de datos vía peticiones REST al backend
  - Componentes reutilizables

  ### 3. Blog Full‑Stack (React + MERN)
  **Descripción**: Blog con interfaz React y backend MERN que permite CRUD de publicaciones.
  **Tecnologías**:
  - React + Hooks + Context
  - Node.js + Express + MongoDB + Mongoose
  **Características**:
  - Crear, leer, actualizar y borrar posts
  - Validación y manejo de errores
  - Comunicación front‑back mediante API REST

  ### 4. Red Social con Autenticación JWT (MERN)
  **Descripción**: Red social sencilla con registro/login, publicación de posts, comentarios, sistema de
  seguidores, y autenticación segura.
  **Tecnologías**:
  - React + Context/Reducers
  - Node.js + Express + MongoDB + Mongoose
  - JSON Web Tokens para autenticación
  **Características**:
  - Registro, login y logout con JWT
  - Protección de rutas en front‑end
  - Gestión de sesiones y autorización
  - CRUD de publicaciones y comentarios
  - Relación entre usuarios (seguidores/seguidos)
